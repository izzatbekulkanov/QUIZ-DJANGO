# Quiz Word Helper Browser Extension

Bu qo'shimcha `Quiz` sahifasi bilan foydalanuvchi kompyuteridagi `Word yordamchisi` orasida aloqa qiladi.

## Qachon kerak bo'ladi

- Sayt `https` orqali ochilganda
- Brauzer `Word yordamchisi` bilan to'g'ridan-to'g'ri ishlashga yo'l qo'ymaganda
- O'qituvchi katta Word faylini bir bosishda olishni xohlasa

## O'rnatish

1. `quiz_word_helper` papkasini kompyuterga saqlang
2. Chrome yoki Edge brauzerida `Extensions` sahifasini oching
3. `Developer mode` ni yoqing
4. `Load unpacked` ni bosing
5. Shu `quiz_word_helper` papkasini tanlang

Shundan keyin sahifani yangilang.
